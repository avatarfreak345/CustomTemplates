How to add new setting

test

1. Add setting in ampconfig.json
2. Add config location to start.sh (ie. GENERAL_XML="$CFG_DIR/GeneralSettings.xml")
3. Add variable in start.sh (ie. SOME_SETTING_HERE="")
4. Add command line arg to start.sh (ie. --server-bin SERVER_BIN="${2:-}" shift 2 ;;)
5. Add line to export new variable(s) (ie. EXPORT SOME_SETTING_HERE)
6. Map config file to variable (ie. gameplay_xml_path = Path(sys.argv[X]))
7. Set config varible to default generated by Luna
8. Add line to read through new variables
9. Set new variables in settings variable
10. Add loop to apply new variables
11. Write out changes (ie. general_xml_path.write_text(general_text, encoding='utf-8'))
